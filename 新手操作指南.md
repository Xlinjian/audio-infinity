# 新手操作指南：让录音更清晰（Audio无限+ 1.0.0，无需任何技术基础）

> 本指南假设你：只会用浏览器、会点鼠标，没装过任何插件。
> 目标：在百度网盘 / B站 / **CCtalk（沪江网校/直播课堂）** / 本地文件里**直接播放**，一键让声音变清晰、降噪、人声变大，还能调「模糊↔清亮」、「环绕音效」，并用**高级均衡器**精细塑形。

> 本版本（1.0.0）为**纯音频增强版**，不含实时字幕功能。需要字幕能力请使用 2.0 版本。

---

## 一、这个工具能帮你做什么（大白话）

- 你有一个录音存在**百度网盘**，或一段视频在 **B站**，或电脑里一个 **mp3**，但声音小、有杂音、听不清人说话。
- 装好这个叫 **「Audio无限+」** 的小插件后，在播放时它会自动帮你：
  - **降噪**：去掉环境杂音、电流声；
  - **人声增强**：让说话声更清楚、更大；
  - **音量均衡**：忽大忽小的声音变得平稳；
  - **低频/高频优化**：去掉闷闷的感觉，声音更通透。
- 还能自己动手调几样东西（见后文）：
  - **人声清晰度**：一条滑块，往左更暖更厚，往右更亮更通透；
  - **环绕音效**：一条滑块，把声音变「宽」、更有空间感，适合音乐。
- 整个过程**不用下载、不用转码、不用学任何软件**，点一下开关就好。

## 二、第一次安装（一次性操作，装过就不用再装）

你电脑上应该已经有一个文件夹叫 `bilibili-audio-booster-1.0`（里面有 `manifest.json` 等文件）。如果只有压缩包 `Audio无限+_1.0.0.zip`，先**右键 → 解压到当前文件夹**。

> 如果你换了一台新电脑，需要把这个文件夹复制过去再装一次。

**第 1 步：打开插件管理页面**
1. 打开 Chrome 浏览器（注意：必须是电脑版 Chrome，手机不行）。
2. 在上方网址栏里**点一下**，把下面这串地址**整个复制粘贴**进去，然后按回车：
   ```
   chrome://extensions/
   ```

**第 2 步：打开“开发者模式”**
- 看页面**右上角**，有一个写着「开发者模式」的开关，点一下让它变成**蓝色/打开**状态。

**第 3 步：加载插件**
1. 点击页面左上角的「**加载已解压的扩展程序**」按钮。
2. 在弹出的窗口里，找到并**选中**那个 `bilibili-audio-booster-1.0` 文件夹，点「选择文件夹」。
3. 页面上出现一个叫「Audio无限+」的卡片，就说明装好了。

> 小提示：装好后浏览器右上角会出现一个新图标（紫色圆 + 白色音波）。如果没看到，点地址栏右侧的「拼图」图标 🧩，在里面找到「Audio无限+」并**点一下旁边的图钉 📌**，让它常驻显示。

## 三、日常使用（每次听录音都这样操作）

**第 1 步：打开你的录音**
1. 登录 `pan.baidu.com`（百度网盘网页版）。
2. 找到你的录音文件（常见的如 `.mp3`、`.m4a`、`.wav`）。
3. **双击**这个文件，它会在网盘里弹出播放器开始播放。

**第 2 步：一键开启增强**
1. 点击浏览器右上角的「Audio无限+」图标，会弹出一个小窗口。
2. 在最上面的「音频优化」右边，把**开关拨到右边（变成紫色）**。

**第 3 步：点一下播放器（很重要！）**
- 由于浏览器的安全规则，**第一次开启后，请用鼠标在录音播放器上点一下**（比如点暂停再播放，或点一下进度条）。
- 这一步只需要做一次。之后小窗口里「引擎」会显示「运行中」。

**第 4 步：选一个效果（推荐“人声优先”）**
- 在小窗口的「音效预设」里，点 **「人声优先」**（最适合录音/会议/访谈）。
- 想让人声更亮更通透，把 **「清晰度」滑块往右拉**；想更暖更厚，往左拉（中间 = 原预设参数）。
- 听音乐会，可以把 **「环绕」滑块往右拉**，声场更开阔。
- 如果觉得整体声音太大或太小，拖动下面的「主音量」滑块调节（中间 100% 是原始大小）。

**第 5 步：享受清晰声音**
- 现在你听到的就是处理过的声音了。换别的录音文件，效果会自动套用，不用重开。

> **小贴士：每个标签页各管各的。** 每个浏览器标签页有自己独立的开关和设置。切到另一个标签页，弹窗里显示的是那个页的状态，互不干扰。

## 三之二、电脑本地文件怎么用（最省心，推荐）

如果你要处理的是**电脑里**的录音（比如桌面上的 `会议录音.mp3`），用插件自带的「本地音频增强器」页面最方便，不用改任何设置：

**第 1 步：打开本地增强页面**
- 点击浏览器右上角「Audio无限+」图标，在小窗口里点 **「打开内置本地播放器」** 按钮。
- （它会新开一个标签页，地址类似 `chrome-extension://.../player/local-player.html`）

**第 2 步：选择你的文件**
- 在新页面里点 **「选择本地文件」**，挑一个音频（mp3 / wav / m4a / flac 都行，也支持带声音的视频）。
- 嫌麻烦的话，也可以**直接把文件拖进**页面里那个虚线框。

**第 3 步：一键增强**
1. 把最上面的「音频优化」开关拨到紫色。
2. 选一个预设，录音/会议建议用 **「人声优先」**。
3. 用「清晰度」滑块调模糊/清亮，用「环绕」滑块加宽声场。
4. 觉得不够响，就拉一下「主音量」滑块。
5. 直接点播放，就能听到变清晰的声音了。

> 这个页面在浏览器里**实时处理**，**不会改动你电脑上的原文件**，也不会上传到任何地方，放心用。

**另一种方法（直接打开本地文件）**：你也可以在地址栏输入 `file:///C:/Users/你的用户名/Desktop/录音.mp3` 直接打开文件，然后照前面「第 2~4 步」开开关、点一下播放器即可。但这种方式需要先到扩展「详情」里打开「允许访问文件网址」；如果嫌麻烦，**直接用上面的「本地音频增强器」页面就好**。

## 三之三、在任意网页放大音量（最高 6 倍，像 Volume Master）

如果你在**别的网站**看视频（比如 YouTube、官网视频、在线课），声音太小，也可以用本插件把音量**直接放大到最高 600%（6 倍）**：

**第 1 步：打开那个网页并播放视频**
- 比如打开 YouTube 的一个视频，让它正常播放。

**第 2 步：打开“全局音量增强”**
- 点击浏览器右上角「Audio无限+」图标。
- 在弹出的小窗口里，你会看到一张「🌊 全局音量增强」卡片（在普通网站上才会出现）。
- 把「音量增强」开关拨到紫色。插件会自动把放大脚本注入这个网页（不用你额外授权）。

**第 3 步：拉高音量**
- 拖动「放大倍数」滑块，最高能拉到 **600%**（也就是原本音量的 6 倍）。
- 觉得够响了就松手。关掉开关，声音立刻恢复正常大小。

> 这个功能只放大音量、不做降噪/人声处理；它和「站内音频优化」是两套独立能力，不会冲突。

## 三之四、自定义外观与功能（设置，在弹窗内）

想换颜色、浅色/深色，或关掉某个处理模块？点弹窗右上角的 **「⚙️」**——设置会**直接在弹窗里展开为一个可滑动的完整页面**，不会跳到新网页。页面分「外观」「功能模块」几个区块，**再点一次「⚙️」或点「完成」就回到主界面**。

- **主题色**：6 款预设色 + 取色器，全界面强调色实时生效。
- **深浅色**：深色（默认）/ 浅色一键切换。
- **模块开关**：人声增强、降噪、压缩、齿音抑制、空气感、环绕，可逐个关闭。关掉后对应处理变为「透明直通」，不影响其余效果。
- 设置会自动同步，并实时下发到已打开的标签页。

## 三之五、高级均衡器（精细塑形音色）

想要自己精调每个频段的音量（低音更厚、人声更突出、高音更亮等）？用**高级均衡器**：

1. 在弹窗顶部「⚙️ 设置」**左边**的 **「🎛 高级均衡器」图标按钮**（与设置按钮同尺寸）点一下（或在设置页、本地播放器内也有同一面板），会在弹窗内展开面板（**不打开新页面**）。
2. 页面顶部有**实时频率曲线**，你拖滑块时曲线立刻变化，所见即所得。
3. 每一段（频段）都可以单独调：
   - **类型**：峰值 / 低架 / 高架 / 低通 / 高通 / 带通 / 陷波
   - **频率**：20Hz ~ 20kHz（用对数滑块精确选频）
   - **增益**：±12dB（往上加、往下减）
   - **Q 值**：控制「这一段有多宽」（越大越窄越尖）
   - 每段可「旁路」临时关掉、可「✕」删除（至少留 2 段），也可「＋ 添加频段」增加段数。
4. 不想自己调，可点**快速预设**：平直 / 默认 / 暖声 / 明亮 / 人声突出 / 低音增强，一键套用（默认仅 3 段，可「＋ 添加频段」扩展）。
5. 改完**自动保存并立即对所有标签页生效**，无需刷新页面。

> 高级均衡器和「音效预设 / 清晰度 / 环绕」不冲突：EQ 在最前面做整体音色塑形，预设负责人声与降噪，叠在一起效果更好。

## 四、常见问题（遇到问题看这里）

**问：开了开关但声音没变化？**
- 先刷新一下网盘页面，重新双击播放录音，再点一下播放器，然后开开关。
- 看小窗口里「引擎」是否显示「运行中」；若是「需刷新页面」，刷新即可。

**问：在不同网页显示的内容不一样？**
- 这是正常的：在 B站/网盘/本地播放页会显示「音频优化」卡片；在其它普通网页会显示「全局音量增强」卡片；两种能力独立、不冲突。

**问：网盘里双击没法播放 / 提示不支持？**
- 有些格式网盘不支持在线播放。这种情况下，请先把文件**下载到电脑**，用本地音频增强器页面播放处理。本插件只负责“在线播放时实时增强”。

**问：CCtalk 没反应 / 媒体只显示 1？**
- 确认在 `*.cctalk.com` 的播放页，并点开「音频优化」开关、点一下播放器激活引擎。CCtalk 播放器常在 iframe 中，本插件会**跨 iframe 自动聚合所有播放器的媒体数**，多播放器页面会显示正确总数。
- 若仍只显示 1：检查是否页面确实只有一个可识别的 `<video>`/`<audio>`（部分页面用 canvas/WebRTC 传输声音，无法被识别）。

**问：高级均衡器改了没声音变化？**
- 确认「启用均衡器」已开，且对应频段增益不为 0；改完所有标签页即时生效，无需刷新。

**问：关闭音频增强后，声音反而发闷 / 不通透？**
- 关闭「音频优化」时，声音走「真正旁路」（直接通向扬声器），原声 100% 保真，不再发闷。若仍发闷，请刷新一次页面、重新加载扩展即可生效。

**问：会影响网盘上传下载吗？**
- 不会。插件只在你**播放**时处理声音，不改变文件本身，也不会上传你的任何内容。

**问：换电脑/重装浏览器后？**
- 插件需要重新按「二、第一次安装」装一次。你的开关设置会随 Chrome 账号云同步，但插件文件要重新加载。

## 五、重要说明（请务必读）

- **这是“边听边增强”**：声音只在你用浏览器播放时变清晰，**不会修改网盘/本地里原来的文件**。
- 如果你需要把“处理好的录音”**保存成新文件**（比如发给别人），本插件做不到“另存为”，请用本地音频增强器播放，或用电脑上的音频编辑软件导出。
- 全部工具免费，不产生任何费用。
